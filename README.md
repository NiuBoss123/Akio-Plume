# MisakaAkio Website (Plume)

御坂秋生的個人網站（基於 Plume 的版本）。

Powered by VuePress & vuepress-theme-plume

## 安裝

```sh
pnpm i
```

## 相關指令

```sh
# 啟動開發伺服器
pnpm docs:dev
# 建構生產環境
pnpm docs:build
# 在本地預覽生產版本
pnpm docs:preview
# 更新 vuepress 和主題
pnpm vp-update
```
